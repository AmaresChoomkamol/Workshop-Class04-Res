# ARSENAL STOCKPILER
### อาวุธที่ได้ :
- แก็งของเราได้รับอาวุธใหม่ก็คือ **เมาส์ ที่ทำจากการกระดาษห่อโรตีเคลือบ radioactive ที่มี core หลักคือ DEMON CORE☢️❗❗❗**

### การใช้งานในภารกิจ
- ใช้เพื่อจบสงครามในเวลาที่คับขัน และเมื่ออยู่ในช่วงคับขัน เราสามารถคลีกกระดาษมาห่อโรตีกินได้ด้วย💀💀💀
 - ฟีเจอร์หลัก : คลิกซ้ายเพื่อระเบิดทุกอย่างด้วยพลังแห่งโรตี โดยที่เราจะต้องคลีด้วยการกดคลิกขวาและใส่โรตีตามที่เราต้องการ หลังจากนั้นก็หอกลับ และเมื่อคลิกซ้ายมันก็จะระเบิด
 - ฟีเจอร์รอง : หากเรากดคลิกขวาแล้วใส่โรตีตามชอบเสร็จ เราสามารถห่อกลับ และใช้เมาส์ที่ 3 หรือ ลูกกลิ้งเลือกตำแหน่งที่ไกลออกไปได้
